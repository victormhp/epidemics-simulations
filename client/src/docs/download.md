## Download Inputs

Download a YAML file containing the inputs to ease the generation of a pre-configured chart. Simply upload this file along with the respective network.

```yaml
algorithm: eventBased
model: fastSIR
tau: 1
gamma: 0.001
rho: 0.005
iterations: 3
tmin: 0
tmax: 20
zoom: on
states: I
```
